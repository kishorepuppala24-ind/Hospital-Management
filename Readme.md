# Smart Emergency & Hospital Management System
## Objective
To implement a Hospital Management System using Queue and Priority Queue in C++.
## Features
- FIFO Appointment Queue
- Emergency Priority Queue
- Severity Based Patient Handling
- Doctor Assignment
- Display Patient Records
## Data Structures Used
- Queue
- Priority Queue
- Heap
## Language Used
C++
## Sample Operations
1. Add Appointment Patient
2. Add Emergency Patient
3. Serve Patient
4. Display Patients
5. Exit
## Applications
- Hospitals
- Emergency Care Systems
- Clinics
